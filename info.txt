1. create virtual env:
> python -m venv venv
> pip install -r requirements.txt

2. add a dependency
> venv\Scripts\activate.bat // needed from pure terminal only (PyCharm does it automatically)
> pip install boto3

3. run a script
> python s3.py

4. add manually exact dependencies to requirements.txt (recommended) or
   grab existing dependencies before commit (it will bring much more transitive dependencies):
> pip freeze > requirements.txt
